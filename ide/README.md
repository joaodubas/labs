# code server with a twist

Run [`code-server`][0] with support to:

1. [asdf (0.7.4)][1]
2. [python (3.7.4)][2]
3. [golang (1.12.9)][3]
4. [erlang/otp (22.0.7)][4]
5. [elixir (1.9.1)][5]
6. [node (12.8.1)][6]

## Run the server

Invoke `docker-compose up -d` and this will start the server.
